Welcome aboard

Congratulations on your first Shake-powered app!

Here's what to do next:

  1. See the README.txt for info about this application skeleton.

  2. Create your models, controllers and views.

    Put them in app/models/, app/controllers/ and app/views/.

  3. Set up your routes and point them to your controllers.

    Routes are set up in app/urls.py. You're seeing this page because you haven't configured any routes yet.

  4. Create your database.

    Run “python manage.py initdb” to create your database. If you're not using SQLite (the default), edit the SQLALCHEMY_URI setting in app/settings/__init__.py with your username and password.

Go ahead, Shake things up!
